# MEMORY.md - 博客封面生成器项目

## 项目概述
博客封面生成器 - 基于 Vite + React + TypeScript 的 Canvas 头图生成应用

## 核心架构

### 技术栈
- **框架**: React 19 + TypeScript 6
- **构建**: Vite 8
- **测试**: Vitest + Testing Library
- **代码质量**: ESLint + Prettier

### 核心模块

#### 1. Canvas 渲染引擎 (`lib/cover/`)
- **renderCover.ts**: 主渲染入口，协调背景渲染和标题绘制
- **seed.ts**: 可复现的随机数生成器（基于 title + style + variant）
- **layout.ts**: 标题排版计算
- **themes.ts**: 主题配置（尺寸、字体预设、主题定义）
- **types.ts**: TypeScript 类型定义

#### 2. 四种风格实现 (`lib/cover/styles/`)
- **tech.ts**: 科技风 - 网格、粒子、光束、芯片面板、HUD面板、六边形等
- **art.ts**: 文艺风
- **business.ts**: 商务风
- **nature.ts**: 自然风

#### 3. 风格推荐引擎 (`lib/recommendation/`)
- 基于关键词匹配的智能风格推荐
- 权重系统 + 语义加分
- 返回置信度等级

#### 4. 历史记录 (`lib/storage/`)
- localStorage 持久化
- 最多保存 12 条
- 支持回放

### UI 组件
- **TitleForm**: 输入标题、选择风格/字体/字号
- **CoverPreview**: Canvas 预览 + 下载
- **HistoryPanel**: 历史记录列表
- **StyleCard**: 风格选择卡片

## 设计亮点

1. **可复现性**: 通过 seed 保证相同输入生成相同封面
2. **变体系统**: variant 实现"换一版"功能
3. **智能推荐**: 基于关键词自动推荐风格
4. **模块化风格**: 科技风使用池化随机选择效果组合
5. **专业尺寸**: 1200×630 标准博客封面比例

## 输出格式
- 下载为高质量 JPG（92% 压缩率）
