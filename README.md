# Blog Cover Generator

一个基于 **Vite + React + TypeScript** 的博客头图生成器，面向博客、公众号、知识卡片和内容分发场景，直接在浏览器里生成 `1200 × 630` 的静态封面图。

![screenshot](public/screenshot.png)

项目核心目标很直接：

- 输入标题，快速生成可用头图
- 支持多种视觉风格，减少重复设计成本
- 允许稳定复现同一张图，也能一键换版
- 本地完成渲染与导出，不依赖服务端

## 功能概览

- 四种内置风格：`科技风` / `文艺风` / `商务风` / `自然风`
- 基于 `title + style + variant` 的稳定 seed，结果可复现
- 支持标题字体切换与字号缩放
- 支持生成纯背景头图，也支持带标题版本
- 支持导出 `JPG` / `PNG` / `WebP`
- 支持将预览图压缩后直接复制到剪贴板
- 本地历史记录回放
- 基于标题关键词的风格推荐

## 使用说明

### 1. 输入标题

- 可以输入完整文章标题
- 也可以留空，直接生成纯背景头图

### 2. 选择标题字体和字号

- 内置多套中文标题字体预设
- 可通过滑杆微调字号比例

### 3. 选择视觉风格

当前支持：

- `tech`：科技风，偏冷色、未来感、数字化
- `art`：文艺风，偏柔和、故事感、质感
- `business`：商务风，偏稳重、专业、结构化
- `nature`：自然风，偏舒展、清爽、生命力

### 4. 智能推荐风格

输入标题后，界面会根据关键词给出推荐风格，并展示匹配强度与推荐原因。

### 5. 生成与换版

- 每次生成都会推进 `variant`
- 同一组 `标题 + 风格 + variant` 会得到稳定结果
- 想保留风格但换一个稳定版本时，只需要重新生成

### 6. 导出与复制

- 支持下载 `JPG` / `PNG` / `WebP`
- 支持直接复制图片到剪贴板
- 复制逻辑会尽量压缩到约 `150KB` 以内，方便粘贴到飞书、Notion、公众号后台之类的场景

## 实现说明

### 渲染模型

项目使用浏览器本地 `Canvas` 渲染：

- 固定输出尺寸为 `1200 × 630`
- 根据风格生成不同背景、装饰元素与排版结构
- 使用 seed 随机数保证随机元素“看起来随机，但实际可复现”

### 可复现策略

封面 seed 来源于：

```text
style + title + variant
```

因此有两个直接结果：

- 相同输入会得到相同封面
- 修改 `variant` 就能稳定生成新的版本

### 本地历史记录

历史记录保存在浏览器 `localStorage`：

- 保存标题、风格、seed、variant
- 支持点击回放
- 支持清空记录

这套设计够轻，没后端也能跑，别整那些没必要的花活。

## 项目结构

```text

  lib/cover/          Canvas 渲染引擎、seed、布局、主题与导出

  lib/storage/        历史记录读写

```
